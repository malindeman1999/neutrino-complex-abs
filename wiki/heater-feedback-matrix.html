Heater Feedback in the Matrix Model

How to include KID-island heater feedback driven by measured phase/quadrature.

Home Theory M Matrix Signal Processing Heater Feedback

Feedback Definition

Let heater-power perturbation be proportional to measured readout quadrature \(Q\):

\[ \delta P_{\mathrm{heater}} = K_Q\,\delta Q,\qquad K_Q\equiv \frac{dP_{\mathrm{heater}}}{dQ}. \]

For a general linear readout projection:

\[ \delta Q = c_r\,\delta r + c_\phi\,\delta\phi, \]

so

\[ \delta P_{\mathrm{heater}} = K_Q(c_r\,\delta r + c_\phi\,\delta\phi). \]

Where It Enters

Only the KID thermal row is modified. In the existing convention, the thermal row (row 3) has electrical-to-thermal terms \(m_{31}\delta r + m_{32}\delta\phi\), then thermal storage/leak terms.

Include heater feedback by adding \(-\delta P_{\mathrm{heater}}\) to that row, which is equivalent to:

\[ m_{31}\rightarrow m_{31}^{(\mathrm{fb})}=m_{31}-K_Q c_r,\qquad m_{32}\rightarrow m_{32}^{(\mathrm{fb})}=m_{32}-K_Q c_\phi. \]

All other matrix entries are unchanged.

Modified Matrix

Using the current 4-state form (\(Y=[r,\phi,T_{KID},T_{ABS}]^T\)):

\[ M_{\mathrm{fb}}(\omega)= \begin{bmatrix} m_{11} & m_{12} & m_{13} & 0\\ m_{21} & m_{22} & m_{23} & 0\\ m_{31}-K_Q c_r & m_{32}-K_Q c_\phi & m_{33} & m_{34}\\ 0 & 0 & m_{43} & m_{44} \end{bmatrix}. \]

If your measured channel is directly the model phase state (\(\delta Q=\delta\phi\)), then \(c_r=0,\;c_\phi=1\), so only one entry changes:

\[ m_{32}\rightarrow m_{32}-K_Q. \]

Sign for Negative Feedback

For current model signs at \(f=0\), a positive absorber-power perturbation gives positive phase response:

\[ (M^{-1})_{\phi,P_{ABS}} > 0. \]

So absorber heating drives \(\delta\phi>0\). To oppose that heating with the heater loop, require \(\delta P_{\mathrm{heater}}<0\) for \(\delta\phi>0\), hence:

\[ K_Q<0 \] (for \(Q\) aligned with model \(\phi\)).

If your readout electronics define \(Q\) with opposite polarity, flip the sign accordingly.

Frequency-Dependent Loop Extension

With loop filter/delay \(F(\omega)\), replace \(K_Q\) by \(K_QF(\omega)\):

\[ m_{31}\rightarrow m_{31}-K_QF(\omega)c_r,\qquad m_{32}\rightarrow m_{32}-K_QF(\omega)c_\phi. \]

This is the standard form for closing a linear feedback path in the frequency-domain model.
