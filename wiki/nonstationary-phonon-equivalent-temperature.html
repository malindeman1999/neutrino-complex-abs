Nonstationary Phonon Noise: Equivalent Temperature

Approximate a pulse-heated absorber with a stationary equivalent temperature for standard optimal filtering.

Home Theory Noise Sources Phonon Nonstationary Phonon Signal Processing

Setup

Assume absorber temperature after an event is

\[ T_A(t)=T_0+\Delta T e^{-t/\tau_p},\qquad \Delta T=T_1-T_0,\quad t\ge 0, \]

while the KID island is actively held near \(T_0\). We approximate the time-varying absorber-KID phonon noise with one stationary equivalent temperature \(T_{\mathrm{eq}}\).

Definition

For a fixed linear estimator/filter \(w(t)\) designed under stationary-noise assumptions, define \(T_{\mathrm{eq}}\) by matching output variance:

\[ \int_0^\infty |w(t)|^2\,T_A(t)^2\,dt = T_{\mathrm{eq}}^2 \int_0^\infty |w(t)|^2\,dt. \]

So

\[ T_{\mathrm{eq}}^2= \frac{\int_0^\infty |w(t)|^2\,T_A(t)^2\,dt} {\int_0^\infty |w(t)|^2\,dt}. \]

This corresponds to the common \(S_P\propto T^2\) thermal-fluctuation scaling for the link-noise term.

Single-Exponential Pulse Case

If the effective stationary optimal-filter weighting is \(|w(t)|^2\propto e^{-2t/\tau_p}\), then:

\[ T_{\mathrm{eq}}^2 = T_0^2+\frac{4}{3}T_0\Delta T+\frac{1}{2}\Delta T^2. \]

Small-pulse limit (\(\Delta T\ll T_0\)):

\[ T_{\mathrm{eq}}\approx T_0+\frac{2}{3}\Delta T. \]

Large-pulse limit (\(\Delta T\gg T_0\)):

\[ T_{\mathrm{eq}}\approx \frac{\Delta T}{\sqrt{2}},\qquad \frac{T_{\mathrm{eq}}}{T_1}\to \frac{1}{\sqrt{2}}. \]

Bounds and Interpretation

Because \(T_A(t)\in[T_0,T_1]\) and \(T_{\mathrm{eq}}^2\) is a positive weighted average of \(T_A(t)^2\):

\[ T_0 \le T_{\mathrm{eq}} \le T_1. \]

So this approximation cannot exceed the worst-case assumption that the absorber stays at \(T_1\) for the full filter window.

The peak contributes more strongly than the long tail, but weighted by \(|w(t)|^2\), not by pulse amplitude alone.
