Area Allocation Rule

Allocate membrane island area as:

\[ A_{mem}=A_{src+cap}+A_{KID}+A_{space}+A_{wire}. \]

This enforces room for:

  • Ho source + superconducting capping footprint
  • KID inductor/capacitor footprint
  • local spacing margin
  • on-island wiring/contacts

Assumed membrane thickness for this estimate:

\[ t_{mem}=0.5\,\mu\mathrm{m}. \]

Example Footprint Budget

TermExample value
\(A_{src+cap}\)\(10\times10=100\,\mu\mathrm{m}^2\)
\(A_{KID}\)\(30\times30=900\,\mu\mathrm{m}^2\)
\(A_{space}\)\(250\,\mu\mathrm{m}^2\)
\(A_{wire}\)\(250\,\mu\mathrm{m}^2\)

Subtotal:

\[ A_{alloc}=1500\,\mu\mathrm{m}^2. \]

With integration margin factor \(m=2.5\):

\[ A_{mem,target}=mA_{alloc}=3750\,\mu\mathrm{m}^2, \]

equivalent to \(\sim61\,\mu\mathrm{m}\times61\,\mu\mathrm{m}\), within a typical 50-100 \(\mu\mathrm{m}\) island range.

Membrane Heat Capacity at 100 mK

For amorphous dielectrics (including amorphous SiN), a standard low-temperature volumetric heat-capacity form is:

\[ c_{V,\mathrm{mem}}(T)\approx aT+bT^3, \] where the linear term is the amorphous/TLS-like contribution and the cubic term is Debye-like phonons.

For first-pass TKID estimates in this wiki, we often use the cubic-dominant simplification \(c_{V,\mathrm{mem}}(T)\approx bT^3\), then evaluate at the operating-point temperature \(T_0\).

Example cubic-only parameterization:

\[ c_{mem}(T)\approx \beta T^3,\quad \beta\approx 2\times10^{-5}\,\mathrm{J/(g\,K^4)}, \]

so at \(100\,\mathrm{mK}\):

\[ c_{mem}(100\,\mathrm{mK})\approx 2\times10^{-8}\,\mathrm{J/(g\,K)}. \]

With \(\rho_{mem}\approx 3.1\,\mathrm{g/cm^3}\), \(t_{mem}=0.5\,\mu\mathrm{m}\), and \(A_{mem,target}=3750\,\mu\mathrm{m}^2\):

\[ V_{mem}=A_{mem,target}t_{mem}=1.875\times10^{-15}\,\mathrm{m^3}, \]

\[ C_{mem}(100\,\mathrm{mK})\approx 1.16\times10^{-16}\,\mathrm{J/K} \approx 0.72\,\mathrm{eV/mK}. \]

Range for 50-100 \(\mu\mathrm{m}\) square islands (same \(t_{mem}\)):

\[ C_{mem}(100\,\mathrm{mK})\approx 0.48\text{ to }1.92\,\mathrm{eV/mK}. \]

Code note: the current Python model uses a fixed input \(c_{V,\mathrm{mem}}\) at the chosen operating point (typically \(T_0\)); it does not yet re-evaluate \(c_{V,\mathrm{mem}}(T)\) dynamically versus temperature.
